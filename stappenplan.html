Marty's HTML/CSS Stappenplan

HTML:

1. Maak een HTML bestand aan en sla deze op via .html
↓
2. Maak een header aan met daar in de titel van je website en voeg je CSS link toe
↓
3. Voeg een body toe met hierin je: titels (h1 t/m h4) paragrafen (p), section's, div's (div), lists (li) etc.
↓
4. Cluster je code doormiddel van class's & id tags
↓
5. Sluit je pagina af met een footer (footer)

CSS:

1. Maak een CSS bestand aan en sla deze op via .css
↓
2. Voorzie de class en id tags van kleur, achtergrondkleur, font, font size, etc.
